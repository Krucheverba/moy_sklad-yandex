   - Товар Takayama НЕ найден

2. **Campaign API - получение товаров по кампаниям:**
   ```
   POST https://api.partner.market.yandex.ru/campaigns/{campaignId}/offers
   ```

   - Товар Takayama НЕ найден ни в одной кампании

3. **Поиск по названию:**

   - Результат: 0 товаров найдено
